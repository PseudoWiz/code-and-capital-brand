### 1. Editorial First

- Typography-driven layouts

- Clear hierarchy guides the eye

### 3. Thoughtful Restraint

- No variations explorer â€” conviction over process
- Tagline as footnote, never headline

## ğŸ“ Tagline Guidelines

The tagline "Where ideas become leverage" follows strict design restraint:
- Smaller than wordmark
- Lighter weight (300)
- Increased tracking (0.05em)
- Neutral grey (#71717A), never indigo
- Sentence case only
- Generous vertical separation (96px)
- Should feel like a footnote, discovered after the wordmark
